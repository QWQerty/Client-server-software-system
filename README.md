# Client-server-software-system
Два микросервиса и предоставление им веб и GUI интерфейсов. Первый микросервис будет работать на локальном  сервере и предоставлять REST API для управления данными о заказах. Второй  микросервис будет развернут на Yandex.Cloud и анализировать данные с  использованием модели линейной регрессии.
