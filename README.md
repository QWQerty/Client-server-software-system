# Client-server-software-system
Два микросервиса с предоставление им веб и GUI интерфейсов. Первый микросервис работает на локальном сервере и предоставляет REST API для управления данными о заказах. Второй  микросервис развернут на Yandex.Cloud и анализирует данные с использованием модели линейной регрессии.
